addValidElementsmethod


public 
						function addValidElements(valid_elements:String):void

Parses a valid elements string and adds it to the schema. The valid elements format is for example "element[attr=default|otherattr]".
Existing rules will be replaced with the ones specified, so this extends the schema.



Parameters





valid_elements:String
String in the valid elements format to be parsed.
